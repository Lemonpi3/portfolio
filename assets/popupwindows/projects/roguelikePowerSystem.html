RogueLike Power System

Project information

  • Category: Unity, 2D
  • Project date: 7 May, 2022

  • Project Github: Link

Generic roguelike powers system for 2D games that can also be easily adapted to 3D. Has lots of customization to create limitless powers using scriptable objects. Includes a small Vampire Survivor's like demo.

You can find a small documentation inside the github repo.
